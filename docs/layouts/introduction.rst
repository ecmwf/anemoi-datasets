#################
 Dataset layouts
#################

The `anemoi-datasets` package supports two types of data organisation
(called "layouts"), both of which are designed to efficiently store and
access large datasets

   -  :ref:`layouts-gridded` datasets are typically meteorological
      fields with regularilty in time and space.
   -  :ref:`layouts-tabular` datasets are typically observations which
      are unstrctired both in time and space.
